# Amazon Clone - Frontend

A fully responsive front-end clone of the Amazon homepage built from scratch. This project was developed to master semantic HTML5 structural layout and advanced CSS3 styling techniques.

## 🚀 Live Demo
You can view the live project here: [https://amazone-clone5.netlify.app/]

## 🛠️ Features
- **Pixel-Perfect UI:** Closely replicates the modern Amazon homepage design.
- **Fully Responsive:** Optimized for desktops, tablets, and mobile screens using CSS Media Queries.
- **Advanced Layouts:** Built using CSS Flexbox and Grid for flexible component alignment.
- **Clean Structure:** Uses semantic HTML tags for better accessibility and SEO structure.

## 🧰 Tech Stack
- **Structure:** HTML5
- **Styling:** CSS3 (Flexbox, Grid, Custom Properties)
- **Icons:** Font Awesome / Google Material Icons (if used, else remove this line)
